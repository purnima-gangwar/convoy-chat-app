# 💬 Convoy Chat App:

A real-time chat application built using the PERN Stack (PostgreSQL, Express, React, Node.js) with Socket.IO for real-time communication and Prisma ORM.

---

## ✨ Features:

- 🔐 Authentication (Login / Signup)
- 💬 Real-time messaging
- 👥 One-to-one chat
- 🟢 Online/Offline status
- ⚡ Instant message delivery
- 📱 Responsive UI

---

## 🛠️ Tech Stack:

- Frontend: React.js, HTML, CSS  
- Backend: Node.js, Express.js, Socket.IO  
- Database: PostgreSQL, Prisma ORM  

---

## ⚙️ Setup:
### 1. Clone repo : git clone https://github.com/purnima-gangwar/Convoy-ChatApp.git

2. Install backend dependencies : 
- cd backend
- npm install

3. Install frontend dependencies : 
- cd frontend
- npm install

5. Setup environment variables (Create a .env file in backend):
- DATABASE_URL= 
- JWT_SECRET= 

# Run the Project :
- Backend: npm run dev
- Frontend : npm run dev
